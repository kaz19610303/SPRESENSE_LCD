# SPRESENSE_LCD

市販されているTFT LCD ディスプレイ(ドライバー IC : ST7735/ST7789)を Spresense に接続するためのインターフェースボードです。  
  
ご利用にあたり、Arduino IDE に次のライブラリを追加してください。  
[Adafruit ST7735 and ST7789 Library](https://www.arduino.cc/reference/en/libraries/adafruit-st7735-and-st7789-library/)  
  
使用可能な LCD Display  
・[ST7789搭載 1.69インチ 280×240 角丸TFT IPSディスプレイ(switch science)](https://www.switch-science.com/catalog/5809/)
